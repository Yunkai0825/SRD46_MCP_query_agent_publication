Using the SRD-46 ammonia system data at about 25 °C, the monammine cumulative constant **log β₁** for the reaction

| Reaction |
|---|
| M²⁺ + NH₃ ⇌ M(NH₃)²⁺ |

ranks as follows:

| Metal ion | log β₁ with NH₃ |
|---|---:|
| Cu²⁺ | 4.15 |
| Ni²⁺ | 2.67 |
| Zn²⁺ | 2.21 |
| Co²⁺ | 2.10 |
| Fe²⁺ | 1.69 |
| Mn²⁺ | 1.00 |

So the order is:

| Rank | Metal |
|---:|---|
| 1 | Cu²⁺ |
| 2 | Ni²⁺ |
| 3 | Zn²⁺ |
| 4 | Co²⁺ |
| 5 | Fe²⁺ |
| 6 | Mn²⁺ |

Equivalently, in increasing stability:

| Increasing log β₁ |
|---|
| Mn²⁺ < Fe²⁺ < Co²⁺ < Zn²⁺ < Ni²⁺ < Cu²⁺ |

**Does it follow the Irving–Williams series?**  
**Yes, broadly yes.** The classic Irving–Williams trend for high-spin first-row divalent ions is:

| Expected trend |
|---|
| Mn²⁺ < Fe²⁺ < Co²⁺ < Ni²⁺ < Cu²⁺ > Zn²⁺ |

The SRD-46 ammonia **β₁** values are consistent with that pattern in the key sense: stability rises from **Mn²⁺ to Cu²⁺**, with **Cu²⁺ highest**, and **Zn²⁺ lower than Cu²⁺**. In these retrieved β₁ data, **Zn²⁺ falls slightly above Co²⁺ but below Ni²⁺**, which is still compatible with the usual “drop after Cu²⁺” behavior rather than a strict monotonic sequence through every intermediate metal.
