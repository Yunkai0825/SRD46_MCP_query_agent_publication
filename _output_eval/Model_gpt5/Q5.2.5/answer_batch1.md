Short answer: SRD‑46 does not report standard electrode potentials for bare redox couples, so it does not provide a numeric E° for the aqueous Cu2+ + 2e− → Cu(s) couple.

What I checked in SRD‑46:
- Searched literature records for “electrode,” “standard electrode,” “potential,” “SHE/NHE” with copper; no entries reporting an E° for Cu2+/Cu(s) were found.
- Queried stability measurements for copper with reaction_type like “redox”; no results.
- SRD‑46 focuses on metal–ligand stability (log β/log K) rather than standard redox potentials.

If you’d like the standard value, it’s tabulated in standard electrode potential tables (e.g., IUPAC/CRC; also available via other NIST resources), but it isn’t contained in SRD‑46. I can retrieve it from a non‑SRD‑46 source on request.

Source: NIST SRD‑46 (Speciation/Stability Database) – no Cu2+/Cu(s) electrode potential data present.
