## Log β₁ for M²⁺–NH₃ (25 °C, I = 0, water) — NIST SRD-46

| Metal | log β₁ | Rank |
|-------|--------|------|
| Mn²⁺ | 1.00 | 6 (weakest) |
| Fe²⁺ | 1.40 | 5 |
| Co²⁺ | 2.11 | 4 |
| Zn²⁺ | 2.37 | 3 |
| Ni²⁺ | 2.81 | 2 |
| Cu²⁺ | 4.31 | 1 (strongest) |

### Observed ordering

**Mn²⁺ < Fe²⁺ < Co²⁺ < Zn²⁺ < Ni²⁺ < Cu²⁺**

### Comparison with the Irving–Williams series

The classic Irving–Williams series predicts:

> **Mn²⁺ < Fe²⁺ < Co²⁺ < Ni²⁺ < Cu²⁺ > Zn²⁺**

The SRD-46 ammonia data **largely conform** to this expectation:

| Feature | Irving–Williams prediction | SRD-46 result | Match? |
|---------|---------------------------|---------------|--------|
| Monotonic rise Mn → Cu | Mn < Fe < Co < Ni < Cu | ✔ Observed exactly | ✅ |
| Cu²⁺ is the strongest | Cu²⁺ at the top | log β₁ = 4.31, well above all others | ✅ |
| Zn²⁺ drops below Cu²⁺ | Cu > Zn | 4.31 ≫ 2.37 | ✅ |
| Zn²⁺ below Ni²⁺ | Ni > Zn | 2.81 > 2.37 | ✅ |
| Zn²⁺ relative to Co²⁺ | Often Co < Zn or Co ≈ Zn | 2.11 < 2.37 (Zn slightly above Co) | ⚠️ Minor deviation |

### Chemistry insight

The data beautifully illustrate the Irving–Williams trend, which arises from the interplay of ionic radius, ligand-field stabilisation energy (LFSE), and the Jahn–Teller distortion that uniquely favours Cu²⁺ (d⁹). The dominant features — a steady rise from Mn²⁺ (d⁵, zero LFSE) through Ni²⁺ (d⁸, maximum octahedral LFSE among the series) to the Cu²⁺ peak, followed by a drop at Zn²⁺ (d¹⁰, zero LFSE) — are all clearly present. The only minor nuance is that Zn²⁺ sits slightly *above* Co²⁺ rather than below it; this is occasionally observed with small monodentate N-donors and reflects the flexible coordination geometry of the d¹⁰ ion, which can adopt a tetrahedral arrangement that is particularly favourable for ammonia.

*Source: NIST Standard Reference Database 46 (Critically Selected Stability Constants of Metal Complexes). All values at 25 °C, I = 0 mol L⁻¹, aqueous solution.*
