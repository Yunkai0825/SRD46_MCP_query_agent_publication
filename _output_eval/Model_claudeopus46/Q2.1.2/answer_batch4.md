# Ranking First-Row Divalent Transition Metals by log β₁ with Ammonia

## Data from NIST SRD-46

All values below are for the equilibrium **M²⁺ + NH₃ ⇌ M(NH₃)²⁺** (log β₁) measured in **water, I = 2.0 M NH₄NO₃, T ≈ 25 °C**. I report the most frequently occurring (modal) value at 25 °C for each metal. **No Fe²⁺ data were returned** from the database for this system — Fe²⁺–NH₃ complexes are notoriously difficult to study because of oxidation and hydrolysis, and SRD-46 apparently lacks entries under these conditions.

| Metal | log β₁ (modal, 25 °C) | Range of reported values |
|-------|----------------------|--------------------------|
| Mn²⁺ | **1.0** | 0.6 – 1.0 |
| Co²⁺ | **2.1** | 1.6 – 2.1 |
| Zn²⁺ | **2.4** | 2.0 – 2.4 |
| Ni²⁺ | **2.8** | 2.2 – 2.8 |
| Cu²⁺ | **4.2** | 3.6 – 4.2 |

### Ranked order (weakest → strongest):

> **Mn²⁺ (1.0) < Co²⁺ (2.1) < Zn²⁺ (2.4) < Ni²⁺ (2.8) < Cu²⁺ (4.2)**

## Comparison with the Irving–Williams Series

The classic **Irving–Williams series** predicts:

> Mn²⁺ < Fe²⁺ < Co²⁺ < Ni²⁺ < **Cu²⁺** > Zn²⁺

| Feature | Irving–Williams prediction | SRD-46 ammonia data | Match? |
|---------|---------------------------|---------------------|--------|
| Mn²⁺ weakest | ✓ | ✓ Mn²⁺ has the lowest log β₁ | ✅ |
| Co²⁺ < Ni²⁺ | ✓ | ✓ 2.1 < 2.8 | ✅ |
| Cu²⁺ is the maximum | ✓ | ✓ Cu²⁺ = 4.2 (highest) | ✅ |
| Zn²⁺ < Cu²⁺ | ✓ | ✓ 2.4 < 4.2 | ✅ |
| Ni²⁺ > Zn²⁺ | ✓ | ✓ 2.8 > 2.4 | ✅ |
| Fe²⁺ between Mn²⁺ and Co²⁺ | Expected | No data available | — |

### Verdict

**Yes — the ammonia log β₁ values follow the Irving–Williams series.** The observed order Mn²⁺ < Co²⁺ < Zn²⁺ < Ni²⁺ < Cu²⁺ matches the predicted trend perfectly (with Fe²⁺ absent from the dataset). Cu²⁺ sits at the apex as expected, and the characteristic drop to Zn²⁺ is clearly seen.

## Chemistry Insight

The Irving–Williams ordering arises from a combination of ionic radius, ligand-field stabilisation energy (LFSE), and the Jahn–Teller distortion of Cu²⁺ (d⁹). Ammonia is a moderate-field, pure σ-donor ligand, so LFSE effects are cleanly expressed. The particularly large jump from Ni²⁺ (2.8) to Cu²⁺ (4.2) — a factor of ~25 in K₁ — reflects the additional Jahn–Teller stabilisation that elongates the axial bonds while strengthening equatorial Cu–N bonds. The absence of Fe²⁺ data is typical: the Fe²⁺–NH₃ system is experimentally challenging because Fe²⁺ is easily oxidised to Fe³⁺ in ammoniacal solutions, and the complex is relatively weak.

*Source: NIST Standard Reference Database 46 (Critically Selected Stability Constants of Metal Complexes).*
