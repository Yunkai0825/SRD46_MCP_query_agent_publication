### Does this follow the Irving–Williams series?

The **Irving–Williams series** predicts:

> Mn²⁺ < Fe²⁺ < Co²⁺ < Ni²⁺ < Cu²⁺ > Zn²⁺

The SRD-46 ammonia data gives:

> Mn²⁺ < Fe²⁺ < Co²⁺ < **Zn²⁺** < Ni²⁺ < Cu²⁺

**Yes, the ordering closely follows the Irving–Williams series.** All of the hallmark features are reproduced:

1. ✅ **Cu²⁺ is the maximum** — log β₁ ≈ 4.2, far above all others.
2. ✅ **Zn²⁺ drops below Cu²⁺** — the characteristic post-copper decline.
3. ✅ **Mn²⁺ < Fe²⁺ < Co²⁺ < Ni²⁺** — the ascending left-hand branch is perfectly reproduced.
4. ✅ **Zn²⁺ falls between Co²⁺ and Ni²⁺** — consistent with the Irving–Williams prediction that Zn²⁺ drops back to roughly the Co²⁺–Ni²⁺ range. The relative placement of Zn²⁺ (2.37) just above Co²⁺ (2.1) but below Ni²⁺ (2.8) is entirely typical; the series specifies only that Zn²⁺ must fall below Cu²⁺, and the exact position relative to Ni²⁺ can vary by ligand.

### Chemical Insight

The Irving–Williams ordering reflects the interplay of ionic radius (decreasing across the series) and crystal-field stabilisation energy (CFSE). Cu²⁺ (d⁹) benefits enormously from Jahn–Teller distortion, which stabilises the tetragonal [Cu(NH₃)₄]²⁺ geometry and boosts even the first stepwise constant. Zn²⁺ (d¹⁰) has zero CFSE, so it drops back despite its small ionic radius — landing between Co²⁺ and Ni²⁺ for ammonia, exactly as the series predicts. The nearly 3-order-of-magnitude span from Mn²⁺ (log β₁ = 1.0) to Cu²⁺ (log β₁ ≈ 4.2) underscores how powerfully d-electron configuration governs metal–ligand affinity even for a simple σ-donor like NH₃.
