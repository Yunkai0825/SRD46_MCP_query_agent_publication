### Step 1 вҖ” Compute the Оұ-coefficient for LВІвҒ» at pH 4

The **side-reaction coefficient** Оұ_L(H) accounts for the fraction of total dissolved malonate that exists as the fully deprotonated form LВІвҒ» at a given pH:

$$\alpha_{L^{2-}} = \frac{K_{a1} K_{a2}}{[\text{H}^+]^2 + K_{a1}[\text{H}^+] + K_{a1}K_{a2}}$$

With pH = 4 вҶ’ [HвҒә] = 10вҒ»вҒҙ M, KaвӮҒ = 10вҒ»ВІВ·вҒ¶вҒө, KaвӮӮ = 10вҒ»вҒөВ·ВІвҒ·:

| Term | Value |
|---|---|
| [HвҒә]ВІ | 10вҒ»вҒё = 1.000 Г— 10вҒ»вҒё |
| KaвӮҒВ·[HвҒә] | 10вҒ»ВІВ·вҒ¶вҒө Г— 10вҒ»вҒҙ = 10вҒ»вҒ¶В·вҒ¶вҒө = 2.239 Г— 10вҒ»вҒ· |
| KaвӮҒВ·KaвӮӮ | 10вҒ»ВІВ·вҒ¶вҒө Г— 10вҒ»вҒөВ·ВІвҒ· = 10вҒ»вҒ·В·вҒ№ВІ = 1.202 Г— 10вҒ»вҒё |
| **Denominator** | 1.000Г—10вҒ»вҒё + 2.239Г—10вҒ»вҒ· + 1.202Г—10вҒ»вҒё = **2.459 Г— 10вҒ»вҒ·** |
| **Numerator** | 1.202 Г— 10вҒ»вҒё |
| **Оұ_LВІвҒ»** | 1.202Г—10вҒ»вҒё / 2.459Г—10вҒ»вҒ· = **0.04888** |

$$\log \alpha_{L^{2-}} = \log(0.04888) = \mathbf{-1.311}$$

This means at pH 4, only ~4.9% of dissolved malonate is in the reactive LВІвҒ» form; the rest is tied up as HLвҒ» (dominant) or HвӮӮL.

### Step 2 вҖ” Compute the Conditional Stability Constant

The conditional constant KвҖІ is defined for the reaction:

$$\text{Cu}^{2+} + L_{\text{total}} \rightleftharpoons \text{CuL}$$

where L_total = [LВІвҒ»] + [HLвҒ»] + [HвӮӮL]. It relates to the thermodynamic constant by:

$$\log K' = \log K + \log \alpha_{L^{2-}}$$

$$\log K' = 5.04 + (-1.311) = \mathbf{3.73}$$

$$K' = 10^{3.73} \approx \mathbf{5{,}370 \ M^{-1}}$$
