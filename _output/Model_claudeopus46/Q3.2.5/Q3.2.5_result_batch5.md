The ~1.1 log-unit gap is significant but not surprising. The discrepancy arises because:

1. **Non-linear activity corrections:** The Debye-Hückel framework predicts that log K should increase steeply as I → 0 for a reaction involving highly charged species like Zn²⁺ and EDTA⁴⁻ (a 2+/4− system). A simple linear fit in √I underestimates this curvature — the true correction is much steeper at very low I.

3. **The I = 0 database value (log K = 18.0)** likely comes from a careful thermodynamic analysis using rigorous activity coefficient models applied to experimental data, which correctly accounts for this non-linearity.

In summary, the finite-I data show the expected trend of **decreasing log K with increasing ionic strength**, and the database's I = 0 value of **18.0** is consistent with a properly corrected extrapolation — the naïve linear extrapolation simply underestimates the true zero-ionic-strength value due to the strong non-linear charge effects in this system.
